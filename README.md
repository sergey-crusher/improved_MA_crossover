# improved_MA_crossover
Улучшенное пересечение MA

Главные изменения, помимо улучшения визуализации стратегии:
- Стоп-лосс (для контроля рисков);
- Минимальная цена при которой сработает мёртвый крест, чтобы сделки не закрывались «бездумно в минус»;
- Принудительный тейк-профит, чтобы если цена улетела вверх, но пересечение не произошло, зафиксировать прибыль (результат при котором можно смело выйди из позиции, без сожалений).
